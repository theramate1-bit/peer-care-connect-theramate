# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

- Code of Conduct
- Contributing guidelines

### Changed
- Improved project organization
- Enhanced documentation structure

### Security
- Security audit and improvements
- Removed hardcoded secrets from codebase

## [1.0.0] - 2025-02-09

### Added
- Initial release of Peer Care Connect / Theramate platform
- Credit-based therapy exchange system
- Real-time messaging between clients and therapists
- Professional verification system
- Advanced scheduling with recurring sessions
- Location-based therapist matching
- Stripe payment integration
- SOAP notes and treatment documentation
- Client progress tracking
- Practice analytics dashboard
- Multi-role authentication (client, practitioner, admin)
- Marketplace for practitioner discovery
- Booking system with calendar integration
- Real-time notifications
- File upload for qualifications
- Review and rating system

### Technical
- React 18 with TypeScript
- Vite build tooling
- Supabase backend (PostgreSQL + Edge Functions)
- Tailwind CSS + Shadcn/ui components
- Playwright E2E testing
- Jest unit testing
- Comprehensive test coverage
