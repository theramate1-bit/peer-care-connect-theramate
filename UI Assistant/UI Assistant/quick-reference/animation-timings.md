# Animation Timings - Quick Reference

## Durations

- **Micro-interactions:** 0.1s to 0.3s
- **Hover effects:** 0.2s to 0.3s
- **Scroll animations:** 0.6s to 1s
- **Page transitions:** 0.3s to 0.5s

## Easing

- **Ease-out:** Most common (starts fast, ends slow)
- **Ease-in-out:** Smooth both ways
- **Spring:** Bouncy, playful feel
- **Linear:** Constant speed (rare)
