# Spacing System - Quick Reference

## Standard Scale

- **4px:** Tight spacing
- **8px:** Small spacing
- **16px:** Standard spacing
- **24px:** Medium spacing
- **32px:** Large spacing
- **48px:** Extra large spacing
- **64px:** Maximum spacing

## Section Spacing

- **Small:** 48px top/bottom
- **Medium:** 80px top/bottom
- **Large:** 120px top/bottom
