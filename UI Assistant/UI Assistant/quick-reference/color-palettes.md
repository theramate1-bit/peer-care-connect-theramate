# Color Palettes - Quick Reference

## Dark Mode

- **Background:** #000000 or #0A0A0A
- **Text:** #FFFFFF
- **Accent:** #007AFF
- **Gray Text:** #CCCCCC

## Light Mode

- **Background:** #FFFFFF or #FAFAFA
- **Text:** #000000 or #1A1A1A
- **Accent:** #007AFF
- **Gray Text:** #666666

## Semantic Colors

- **Success:** #34C759 (green)
- **Error:** #FF3B30 (red)
- **Warning:** #FF9500 (orange)
- **Info:** #007AFF (blue)
