# Typography Scales - Quick Reference

## Standard Sizes

- **Title:** 32px (desktop), 24px (mobile)
- **H1:** 28px (desktop), 24px (mobile)
- **H2:** 24px (desktop), 20px (mobile)
- **Body:** 18px (desktop), 16px (mobile)
- **Small:** 14px
- **Caption:** 12px

## Line Heights

- **Headings:** 1.2 to 1.4
- **Body:** 1.5 to 1.75
- **Large Text:** 1.4 to 1.6

## Font Weights

- **Regular:** 400
- **Medium:** 500
- **Semibold:** 600
- **Bold:** 700
