# Breakpoints - Quick Reference

## Standard Breakpoints

- **Mobile:** 375px - 767px
- **Tablet:** 768px - 1439px
- **Desktop:** 1440px+

## Usage

Use these breakpoints in media queries to adjust layouts, font sizes, and spacing for different screen sizes.
