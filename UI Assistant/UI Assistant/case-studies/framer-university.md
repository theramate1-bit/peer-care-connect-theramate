# Framer University Case Study

## Overview

Framer University is a modern, high-contrast, dark-themed website that demonstrates advanced Framer interactions and component-based architecture.

## Key Design Patterns

1. **Dark Theme:** Near-black background (#000000 or #0A0A0A) with white text
2. **Sticky Navigation:** Navigation stays visible while scrolling
3. **Scroll Effects:** Subtle parallax and fade-in animations
4. **Component-Based:** Reusable components (Navigation, Hero, Cards, Accordion)
5. **Grid Layout:** 3-column responsive grid for resources
